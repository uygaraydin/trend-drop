# Data Import Guide

You can add manual data to this folder for the agent to use.

## Accepted Formats
- **urls.md** — Extra URL list to scan (one URL per line)
- **trends.csv** — Manual trend data (trend_name, category, source, date)
- **notes.md** — Notes or directives you want to pass to the agent

## Example urls.md
```
https://www.vogue.com/fashion/trends
https://www.pinterest.com/search/pins/?q=fashion+trends+2026
```
